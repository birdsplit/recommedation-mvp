# 카탈로그 신규 판매처 추가 조사 노트 (additions)

작성: additions-agent-2026-07-20 · 대상 규격: 슈퍼싱글(SS) 침대 **프레임**
산출물: `data/catalog-products.additions.csv` (13행), `data/review-checklist.additions.csv` (13행)

기존 사용 가능 18행(일룸/리바트몰/한샘몰/지누스/에이스침대)과 중복되지 않는 **신규 판매처 3곳**에서 SS 프레임 13개를 수집했다.
합산 시 31개 상품 / 8개 판매처로 릴리스 목표(≥30개·≥6곳)를 충족한다. 전 행 `status=hidden`, `availability=unknown`, `image_url` 공란.

## 도구 제약 (중요)
- 본 환경에서 **WebFetch 와 WebSearch 모두 사용 불가**였다(WebSearch는 모델 접근 오류로 매 호출 실패).
- 따라서 검색 없이 **후보 도메인 직접 프로빙 → sitemap.xml / 카테고리 페이지 curl → 상품 링크 파싱 → 개별 상품 페이지 curl** 로 수집했다.
- 데스크톱 UA, 도메인당 요청 간 1초 내외 간격, 로그인·봇우회 없음. 저장 경로: scratchpad/additions.

## 판매처별 수집 방법 메모
- **잉글랜더(Godo)**: EUC-KR/CP949 디코드. 가격은 `set_goods_fixedPrice` hidden input(예 219000.00). 옵션 `매트제외-SS`가 기본(+0), Q/K는 추가금(+40,000~+70,000) → SS는 **프레임 단품(매트 제외)**. `상품정보제공고시` 표에서 제조국·품질보증(1년 무상 A/S)·배송설치비용(수도권 7일/지방 15일, 착불/지역조건부) 확보. 리뷰는 페이지에 표본 노출 없음.
- **소프시스(Cafe24)**: EUC-KR/CP949. `og product:price:amount`=SS 판매가. 배송비 고시(무료/20,000/착불 30,000) 확보. 리뷰는 **성인인증 게이트**로 curl 표본 불가.
- **마켓비(Cafe24)**: UTF-8. meta 가격(VAT 포함 판매가). Cafe24 `상품정보제공고시`에서 제조사·**원산지**·배송비·제품타입(간편조립/방문설치) 확보. 기본옵션 '프레임만 구매' → **매트 별도**, 매트리스·조립은 유료 부가옵션. 리뷰 AJAX 로드로 표본 불가.
- **치수 공통 한계**: 세 몰 모두 완성 외경(가로/세로/높이)을 **상세 이미지**로만 제공 → `width_cm/length_cm/height_cm` 전부 공란(추정 금지). 상품명의 `110200`은 매트리스 규격이라 프레임 외경으로 저장하지 않음.

## 커버리지 요약 (기존 18행 대비)
- **수납 방식**: 기존 18행은 none/lift_up/legs_open/drawer 위주였고 **평상형(closed_base)이 없었다**. 이번 추가로 closed_base 3건(소프시스 6910, 잉글랜더 2640·3494)을 신규 확보. drawer 확정 1건(마켓비 39151 서랍형), legs_open 추정 1건(마켓비 6675 데이베드), 수납방식 미확정 5건(잉글랜더 수납 3 + 마켓비 패브릭 2, 검수 필요).
- **가격대**: 110,900 ~ 1,090,000원. 200k~300k·300k+ 및 ~100만원대까지 폭을 넓힘(기존 대비 고가·세트 구간 보강).
- **배송비**: 확정 무료(2), 확정 유료(14,000/20,000/33,000/착불 30,000/100,000), 지역조건부 미확정(잉글랜더 5) — 기존의 무료·미확인 편중을 유료·조건부로 다양화.
- **매트리스 포함**: 프레임+매트 세트 2건(소프시스 6827·6829, mattress_included=true)으로 구성 다양성 추가.
- **원산지**: 대한민국/베트남/중국/말레이시아 혼합.
- **설치**: 유료(소프시스 6117, 마켓비 6675·26956·40844), 포함(마켓비 39151 전국 방문설치), 미확인(나머지).
