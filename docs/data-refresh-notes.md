## 수집 방법

- WebFetch는 이 환경에서 동작 불가(내부 오류)라 사용하지 않았고, 데스크톱 UA로 `curl`을 사용했습니다(요청 ≤1회/초, 각 상품 페이지 1회).
- 상품 페이지 18개 + 에이스침대 제조사 사양 페이지 2개(치수 재확인용) = 총 20회 요청, 전부 HTTP 200.
- 저장 HTML은 스크래치패드에서 파싱했습니다(모든 페이지 UTF-8). 페이지에 없는 값은 추정하지 않고 빈칸/`unknown`으로 남겼습니다.
- 상업정보(`commercial_verified_at`)·사양(`spec_verified_at`) 확인일은 2026-07-20으로 기록했습니다.
- 근거 신뢰도(`*_confidence`)는 **오늘 해당 페이지에서 직접 본 값만 `confirmed`**, 7/12 관측을 유지했거나 일반 정책에서 유추한 값은 `estimated`로 표기했습니다.
- 이미지 사용권 미확인이라 `image_url`은 전 행 공백입니다.

## availability 판정 기준(이번 재검증에서 변경)

명시적·기계판독 가능한 재고 신호가 있는 판매처만 `in_stock`으로 올렸습니다.

- `in_stock`(9행): **일룸**(JSON-LD `schema.org/InStock`), **한샘몰**(embedded `__NEXT_DATA__`의 `soldOutYn=false`·`stockQty=999`).
- `unknown`(9행): **리바트몰·지누스·에이스침대** — 판매가와 구매버튼은 노출되고 품절 표시는 없으나, 명시적 재고 신호를 curl로 확인할 수 없어 유지. 옵션·지역별 최종 재고는 사람 검수에서 확정.
- `out_of_stock`: 없음(404·품절 페이지 없음).

## 판매처별 결과

### 일룸 (5행) — 가장 공개에 가까움
- 완전 서버 렌더링 + JSON-LD(가격·`InStock`·평점). 5개 **가격 모두 7/12와 동일**(899,000 / 1,009,000 / 452,000 / 2,490,000 / 649,000).
- 배송: 상세의 "영업일 기준 7~10일" 반영(초안 10일→7~10). 전 상품 무료 설치배송(전문시공팀)·배송일 지정 가능.
- 치수: **HBA501101**은 초안 공백이었으나 라벨 스펙(가로1141·세로2170·높이1100.5mm)으로 채움. IBM0021A는 라벨 스펙(1280·2225·887mm) 재확인. HBA411121은 라벨 없는 `1105*986*2191.5`를 물리적 범위로 폭110.5·길이219.15·높이98.6cm 매핑(사람 재확인 권장). **HSSE270은 완성 외경 미표기라 치수 공백 유지.**
- 정책: 반품(배송 후 취소 불가·품목별 반품비)·파손(정상 설치품 A/S)·보증(소비자분쟁해결기준 1년 무상 A/S) 요약 확보.
- 리뷰: HTML엔 집계 평점만(본문은 XHR) → 표본 미수집. 남은 공개 조건은 리뷰 표본 + 이미지 권리(+HSSE270 치수).

### 리바트몰 (4행)
- 서버 렌더링(JSON-LD 없음), 가격·치수·정책 모두 HTML에 존재. **가격 4개 모두 동일**(비회원가 573,000 / 490,000 / 795,000 / 617,000). 회원가는 별도(예: P100017510 약 320,800원).
- 치수: 초안 공백이던 3개를 사이즈 필드로 채움 — P100017510(1141·2049·**1200**mm, 하우스프레임형 높이), P200199572 SS(1140·2154·1050mm), P200183576(1134·2130·높이 810~960 가변→height 공백). **P100026884는 정식 외경 표기 없음(본문 수치는 성인 SS 사용공간 안내)라 치수 공백.**
- 배송/정책: 기본 배송비 무료(제주 제외)·"내맘대로 배송지정"(scheduled=true)·직접배송 설치. 배송기간 4~5일은 일반 배송정책이라 `estimated`. 반품(수령 익일 7일 이내·조립 후 단순변심 불가·하자 예외)·회원 3년 무상 A/S 요약 확보.
- 남은 조건: availability(재고) 확정, 리뷰 표본, 이미지 권리. P200199572는 헤드 "중간 수납/서랍" 언급이 있어 수납 구조 확인 필요(현재 storage_type=none 유지).

### 한샘몰 (4행) — 구조적 데이터 공백 큼(브라우저 재검수 필요)
- `store.hanssem.com`/`mall.hanssem.com`은 curl 기준 Next.js JS 셸이지만 **embedded `__NEXT_DATA__`에서 가격·재고 추출 성공**.
- 가격(현재 표시 할인가 dcPrc)·재고: 411763 **345,900→349,900**(20%↓) · 411762 **259,900→252,900**(21%↓) · 505285 359,900(동일, 23%↓) · 724132 **436,000→369,000**(초안은 판매가, 오늘은 표시 할인가 16%↓). 4개 모두 `soldOutYn=false`·`stockQty=999`→`in_stock`.
- 치수·색상·배송비·소재·정책은 이미지/JS 렌더라 HTML에 없어 **공백 유지**(지시대로 추정 금지). 724132만 7/12 관측 외경(115.6·214.7·91.2)·소재·설치를 `estimated`로 유지.
- 남은 조건(전 행): 치수·색상·배송비·반품/파손/보증 요약을 브라우저로 확보 + 리뷰 표본(평점 4.8~4.9, 30~412건). 가격·재고만 확정된 상태.

### 지누스 (3행)
- 서버 렌더링. 가격은 embedded JSON, 치수·정책은 HTML.
- **폴 침대(1000000217)**: 최저 판매가 205,000(동일)·SS 113×206×105cm·하부공간 30.6cm·5년 품질보증·1시간 내 조립(공구 동봉)·무료 택배 확인. 하부 30.6cm+다리형→`robot_vacuum_fit=ok`, `self_assembly=easy`. 소재는 페이지가 **"소나무 원목"**(초안 "아카시아"와 상이)라 페이지값 기록+플래그.
- **루미 2개(1084/1082)**: 최저 판매가 499,000 / 374,000(동일)·색상 크림화이트/내추럴오크·"설치배송 서비스만 제공"(엘베 없는 2층↑ 사다리차 등 조건부 추가비용). 정확한 SS 옵션 조합가·완성 외경은 옵션 선택 후 확정 필요→`data_confidence=estimated`, 배송비 `unknown`.
- 정책: 반품(수령 7일 이내·단순변심 왕복배송비)·파손(하자 배송비 판매처 부담) 확보. 루미 보증은 페이지 미표기→공백.
- 남은 조건: 폴은 availability·배송기간·리뷰 표본·이미지; 루미는 옵션 조합가·외경·보증까지.

### 에이스침대 (2행)
- 에이스침대몰(판매가/할인가·옵션 JSON) 서버 렌더링 + **에이스침대 제조사 사양 페이지 재요청으로 치수 오늘 재확인**.
- **가격 2개 모두 동일**: BRA1432 판매가 591,000·8% 할인 543,720 / BRA1437 판매가 954,000·8% 할인 877,680.
- 치수 재확인: BRA1432 SS 너비1140·길이2099·높이898mm(114×209.9×89.8) / BRA1437 SS 너비1140·길이2133·높이1008mm(114×213.3×100.8)·소재 우드st·컬러 레귤러오크/월넛내추럴.
- 배송비: 무료배송이나 도서산간 지역별 20,000~150,000원 추가 가능→`shipping unknown`. 프레임 재고 신호는 확인 불가→`availability unknown`.
- 정책: 몰 페이지에서 반품/보증 요약을 깔끔히 추출하지 못해 **공백**(사람 확보 필요).
- 남은 조건: availability·지역별 배송비·반품/보증 요약·리뷰 표본·이미지 권리.

## 리뷰 표본에 관한 공통 사항

전 판매처에서 개별 리뷰 본문은 XHR/JS로 로드되어 curl HTML에는 집계 평점만 존재합니다. 지시된 대체 원칙에 따라 **리뷰 관련 컬럼은 전부 공백으로 두고 사람 검수에 위임**했습니다(`review_risks`도 공백=반복 위험 미노출로 검증 통과). 참고용 집계 평점: 일룸 HSRE01SS 5.0/2 · HSSE270 4.9/68 · HBA501101 4.8/24 / 한샘 411762 4.8/412 · 411763 4.8/188 · 724132 4.9/142 · 505285 4.8/30.
